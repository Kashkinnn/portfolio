  • Browser Concept (Prototype) ACTUAL CONCEPT IS IN PROGRAMMING PROJECTS SECTION

    • - Project PrintPress, A Website concept that processes printing requests from users, students in particular since this is to be implemented in schools and to be processed and handled by accounts with staff access. And also both the student and staff account can be managed using the admin access.

    - Student : Can create print requests, by creating request and providing required information such as the files, print type and where to have it printed. (Print price is already pre determiend by the site).

    - Staff : Can handle print requests made by students, can mark requests as "Processing", or "Completed". Also has inventory mangement dashboard in the actual implementation which can be found in programming project section.

    - Admin : Has full CRUD functionalities, like creating, reading, updating, deleting accounts, inventory and stations.

    - Design thought process and philosophy : I created concept site because of the headache and time consumed of going to print stations just to get stuff printed and encountering long lines. The design is solely based of how I want things to look simple while maximizing functionality and efficiency. And most importantly having less friction as possible.
